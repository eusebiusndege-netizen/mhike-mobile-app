Overview
A web-based application developed to digitize the management of Kenyan Chamas by allowing administrators and members to manage contributions, loans, repayments, and member records.

Features
User authentication
Role-based access
Member management
Loan management
Contribution tracking
Notifications
Dashboard
Reports


Technologies
Node.js
Express.js
MySQL
HTML
CSS
JavaScript
